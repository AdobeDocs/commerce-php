# Magento_CustomAttributeManagement module

The Magento_CustomAttributeManagement module implements user-defined attributes management which provides the ability to manage attributes of customers and their address.

The admin user can manage attributes on the UI level without assistance of a programmer.

The admin user can create new, modify, and remove attributes, control attributes properties and visibility on the frontend.

User defined attributes are attributes which are created by the admin user and are not available out of the box.

These attributes can be deleted from the system after their creation.

System attributes are attributes which cannot be deleted from the system in the usual way and the admin user can edit only their label.

A hidden attribute is an attribute which is hidden from the user on the back-end and on the front-end.
