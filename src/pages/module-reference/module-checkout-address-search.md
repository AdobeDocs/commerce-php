# Magento_CheckoutAddressSearch module

The Magento_CheckoutAddressSearch module extends Magento_Checkout and adds functionality to search customer shipping and billing addresses with the `ui-select` component.

It prevents loading and rendering of all customer addresses on checkout.

This feature is disabled by default and can be configured in the Admin Sales Checkout options. Default limit of addresses after which address search is enabled is 10 and it can be configured in the Admin.

<InlineAlert slots="text" />
The version of this module is 100.4.8.
