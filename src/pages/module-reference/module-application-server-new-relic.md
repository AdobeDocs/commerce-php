# Magento_ApplicationServerNewRelic module

The Magento_ApplicationServerNewRelic module enables reporting for performance and reliability data of the Application Server using the New Relic service.

## Prerequisites

To take advantage of this module, you must have a New Relic account and install the New Relic extension on your environment.

<InlineAlert slots="text" />
The version of this module is 100.4.2.
