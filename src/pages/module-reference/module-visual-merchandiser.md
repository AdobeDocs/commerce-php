# Magento_VisualMerchandiser module

This module allows you to create and merchandise categories quickly and easily with Visual Merchandiser.

Drag-and-drop products into position, or set up 'Smart Categories' based upon attributes.

This module saves you hours merchandising your store.

With Visual Merchandiser, products can be re-organised in seconds – visually. You see the products in place, with images, just like the customer sees them.
