# Magento_ScalableInventory module

_Deprecated since 2.4.2 and will be removed._

This module provides the ability for system extension (CatalogInventory can be configured to work with separate quantity storage).

Extraction of quantity updates to separate storage will guarantee better scalability for Magento, and will allow the main server to be optimized for read operations which will reduce latency.

The version of this module is 100.4.7.
