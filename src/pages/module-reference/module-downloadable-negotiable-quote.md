# Magento_DownloadableNegotiableQuote

The Magento_DownloadableNegotiableQuote module enables downloadable product to be displayed in a negotiable quote in a B2B environment. This module extends Magento_NegotiableQuote and Magento_Downloadable modules.

The Magento_DownloadableNegotiableQuote module provides the following features:

* Display downloadable in a negotiable quote.

* Order downloadable from a negotiable quote.

## Installation details

The Magento_DownloadableNegotiableQuote module has a dependency on the Magento_NegotiableQuote and Magento_Downloadable modules, which must be installed and enabled first. This module does not create any backward incompatible changes. It can be uninstalled or deactivated at any time.
