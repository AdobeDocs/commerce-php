# Magento_AsyncOrderGraphQl module

Magento_AsyncOrderGraphQl module provides type and resolver information for the GraphQl module
to run asynchronous operations on orders. It requires the Magento_AsyncOrder module to be enabled.

<InlineAlert slots="text" />
The version of this module is 100.4.4.
