# Magento_LoginAsCustomerLogging module

The Magento_LoginAsCustomerLogging module provides logging functionality for admin actions performed while using the "Login as Customer" feature. This module helps track and maintain an audit trail of administrative actions taken while impersonating customers.

<InlineAlert slots="text" />
The version of this module is 100.4.9.
