# Magento_ReCaptchaPurchaseOrder module

The Magento_ReCaptchaPurchaseOrder module integrates Google reCAPTCHA protection for purchase order operations in Magento B2B. It adds an additional security layer to prevent automated bot submissions and spam during purchase order placement.

<InlineAlert slots="text" />
The version of this module is 1.0.0.
