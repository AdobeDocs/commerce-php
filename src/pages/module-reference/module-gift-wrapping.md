# Magento_GiftWrapping module

The Magento_GiftWrapping module provides functionality that allows the customer to add gift wrapping to the items purchased from the store as gifts and charge it individually.

The Magento_GiftWrapping module extends the functionality of gift messages by combining gift messages with gift wrapping functionality.

The version of this module is 101.2.8.
