# Magento_GiftCardImportExport module

The Magento_GiftCardImportExport module introduces import and export functionality for GiftCard Product.

This module extends existing functionality of Magento_CatalogImportExport module by adding a new product type.

<InlineAlert slots="text" />
The version of this module is 100.4.6.
