# Magento_AdvancedCatalog module

Magento_AdvancedCatalog module introduces a list of optimizations to allow higher concurrency on product management
operations with immediate update of product data on the frontend and plays as an extension to the indexation logic of
the Magento_Catalog module.

<InlineAlert slots="text" />
The version of this module is 100.4.6.
