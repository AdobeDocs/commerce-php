# Magento_SocialLoginGraphQl module

This module provides social login compatibility.

## Installation

Before installing this module, note that the Magento_SocialLoginGraphQl is dependent on the following modules:

- `Magento_Customer`
- `Magento_CustomerGraphql`
- `Magento_Integration`
- `Magento_Newsletter`
- `Magento_Store`
